# Automated File Organizer

Python script for automatically organizing files in directories by type/category.

## Features
- **Automatic Sorting**: Organize files by extension
- **Category Creation**: Create folders for file types
- **Bulk Operations**: Process multiple files at once
- **File System Safety**: Non-destructive operations
- **Customizable**: Configurable organization rules

## Technology Stack
- Python
- os module
- File system operations
- Path manipulation

## Use Cases
- Downloads folder organization
- Project file cleanup
- Backup organization
- Media file sorting
- Document management

## Typical Organization
- Documents (pdf, doc, docx, txt)
- Images (jpg, png, gif, svg)
- Videos (mp4, avi, mov)
- Audio (mp3, wav, flac)
- Archives (zip, rar, tar)
